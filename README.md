# 아레스 웹 리뉴얼
## 토이 프로젝트
* 프로젝트 멤버 
 * 백엔드 : 한수윤, 김수윤, 양민욱
 * 프론트 : 한수윤, 김수윤
 * 디자인 : 김주은

## 개발 환경
* 백엔드 : JAVA(1.8), Spring boot, JPA, queryDsl
* 프론트 : Vue.js
* 웹서버 : Nginx
* 데이터베이스 : Mysql
* 데이터 스트럭처 : Redis
* 자동배포툴 : Jinkins
* 버전관리 툴 : GitHub
* 업무협업 툴 : Slack

## 기능 목록
* 경기일정
 * 경기일정 등록
 * 경기일정 수정
 * 경기일정 조회
 * 경기 참석 / 불석
 * 참석 / 불참 / 미투표 회원 정보 노출

* 회원관리
 * 회원 조회
 * 회원 수정
  * 일반회원 수정 가능 정보 : 이름, 연락처, 프로필, 나이, 선호 포지션
  * 관리자 수정 가능 정보 : 회원 등급
 * 회원 등급 수정
